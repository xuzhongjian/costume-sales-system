# 主要功能

本系统的功能主要有

1. 商品销售 - sales
   1. 出售 - sale
   2. 退货 - return
   3. 换货 - exchange
2. 商品查询 - items
   1. 商品搜索 - search
   2. 门店查询 - store
   3. 商品查询 - item
3. 货物供应 - stocks
   1. 入库 - in
   2. 出库 - out
4. 门店查询 - stores
   1. 附近门店 - around
   2. 地区门店 - localtion
   3. 门店搜索 - search
5. 数据展示 - display
   1. 数据分析 - analysis
   2. 数据监控 - monitor

另外，用户登录，也是一个很重要的模块。
